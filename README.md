## Overview

Price4U is an eCommerce price tracker that allows users to monitor product prices across various online platforms.

## Technologies Used

- **Framework**: Next.js 15
- **Library**: React.js
- **Styling**: Tailwind CSS
- **Web Scraping**: Bright Data
