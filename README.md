# Price4U

<img src="https://github.com/sulimanbadour1/Price4U/blob/main/public/screens/demo.gif?raw=true" alt="Banner" width="460px"/>

## Overview

Price4U is an eCommerce price tracker that allows users to monitor product prices across various online platforms.

## Technologies Used

- **Framework**: Next.js 15
- **Library**: React.js
- **Carousel** : React Responsive Carousel.
- **Styling**: Tailwind CSS
- **Web Scraping**: Bright Data
